# Stap 10 — Laat je werk zien! 🎉

Je bent klaar! Loop even naar je begeleider en laat je pagina's zien in de browser. Laat zien:

- Je profiel pagina op `/stagiair`
- Je hobby pagina op `/stagiair/hobby`
- De navigatie tussen de pagina's
- De like button in actie
- Hoe het er op mobiel uitziet (F12 → mobiel icoon in DevTools)

---

## Bonus — Als je nog tijd over hebt

- **Dark mode:** Voeg een toggle toe die wisselt tussen licht en donker thema (tip: Tailwind `dark:` prefix)
- **Animatie:** Voeg een simpele CSS animatie toe, bijvoorbeeld een bounce effect op je avatar
- **Responsive:** Test je pagina's op verschillende schermformaten via de browser DevTools (F12 → mobiel icoon)
- **Favicon:** Vervang de standaard favicon in `frontend/app/favicon.ico` door een eigen icoontje

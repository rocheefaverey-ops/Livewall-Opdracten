# Welkom bij het team! 🎉

## Stap 0 — Setup

Voordat we beginnen, gaan we eerst het project draaiend krijgen op je laptop.

1. Open het project in VS Code
2. Installeer de aanbevolen extensies: ESLint, Tailwind CSS IntelliSense, Prettier
3. Ga naar de `frontend` map:
   ```bash
   cd frontend
   ```
4. Installeer de dependencies:
   ```bash
   npm install
   ```
5. Maak je eigen environment bestand aan:
   ```bash
   cp .env.example .env
   ```
6. Start de development server:
   ```bash
   npm run dev
   ```
7. Open [http://localhost:3000](http://localhost:3000) — je zou de standaard Next.js pagina moeten zien

✅ **Klaar?** Laat even zien dat de pagina draait, dan krijg je de volgende opdracht!

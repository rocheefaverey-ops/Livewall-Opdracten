# Stap 1 — Verken het project

Voordat je code gaat schrijven, neem even de tijd om het project te begrijpen. Open de volgende bestanden en lees ze door:

| Bestand | Wat doet het? |
|---------|---------------|
| `frontend/app/layout.tsx` | De root layout — alles wat op elke pagina staat (fonts, html structuur) |
| `frontend/app/page.tsx` | De homepage — dit is wat je ziet op `localhost:3000` |
| `frontend/app/globals.css` | Globale styling en CSS variabelen (kleuren) |
| `frontend/tailwind.config.js` | Tailwind configuratie met custom kleuren |
| `frontend/next.config.ts` | Next.js configuratie (security headers, caching) |
| `frontend/public/` | Map voor statische bestanden (afbeeldingen, icons) |

**Schrijf voor jezelf op:**
- Welke fonts worden er gebruikt?
- Welke kleuren zijn er gedefinieerd als CSS variabelen in `globals.css`?
- Wat is het verschil tussen de `app/` map en de `public/` map?

✅ **Klaar?** Bespreek je antwoorden even kort, dan krijg je de volgende opdracht!

# Quiz App — Introductie 🎯

Je hebt Memory gebouwd, nice! Nu gaan we iets anders maken: een **Quiz App**.

## Wat gaan we maken?

Een interactieve quiz waarbij je:
- Een vraag ziet met meerdere antwoordopties
- Een antwoord selecteert en feedback krijgt (goed/fout)
- Door meerdere vragen navigeert
- Aan het einde je score ziet met een overzicht

## Wat je gaat leren

| Onderwerp | Stap |
|-----------|------|
| Project opzetten en data structuur | Stap 1 |
| Vraag en antwoorden weergeven | Stap 2 |
| Antwoord selecteren en feedback tonen | Stap 3 |
| Navigatie tussen vragen | Stap 4 |
| Score berekenen en resultaat tonen | Stap 5 |
| Timer per vraag toevoegen | Stap 6 |
| Polish: animaties en extra features | Stap 7 |

## Structuur

We bouwen alles in een nieuwe route:
```
frontend/app/(pages)/quiz/
```

## Verschil met Memory

| Memory | Quiz |
|--------|------|
| Alle kaarten tegelijk zichtbaar | Één vraag per keer |
| Klikken op items | Selecteren van opties |
| Matchen van paren | Goed/fout antwoorden |
| Beurten tellen | Score bijhouden |

> **Tip:** Je kunt altijd terugkijken naar je Memory code als je iets vergeten bent. Veel patronen komen terug!

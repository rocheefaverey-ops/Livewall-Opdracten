# Memory Game — Introductie 🧠

Je hebt alle basis-opdrachten afgerond, nice! Nu gaan we iets leukers bouwen: een **Memory Game**.

## Wat gaan we maken?

Een klassiek memory spel waarbij je:
- Een grid met kaarten ziet (met de achterkant naar boven)
- Twee kaarten per beurt omdraait
- Bij een match blijven de kaarten open
- Bij geen match draaien ze weer terug
- Wint als alle paren gevonden zijn
